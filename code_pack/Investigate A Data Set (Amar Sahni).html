Adding additional columns for the release duration based on the release_year (4 Groups as mentioned below)¶

>

  1. 1960 - 1980 : '1960-1980' (Group 1)

  2. 1981 - 2000 : '1981-2000' (Group 2)

  3. 2001 - 2010 : '2001-2010' (Group 3)

  4. 2011 - 2015 : '2011-2015' (Group 4)

Adding additional columns for the run type based on the run time (4 Groups as mentioned below)¶

>

  1. 0-9 : 'very short' (Group 1)

  2. 10-49 : 'short' (Group 2)

  3. 50-299 : 'long' (Group 3)

  4. 300 and Above : 'very long' (Group 4)

In [150]:
# Adding additional row so as to group by release_date duration

mov_df['generation'] = mov_df.release_year.apply(lambda x: '1960-1980' if (x>=1960 and x<=1980) else ( '1981-2000' if(x>=1981 and x<=2000) else ('2001-2010' if(x>=2001 and x<=2010) else '2011-2015')))
In [151]:
# Adding additional row so as to group by run_time duration

mov_df['runtype'] = mov_df.runtime.apply(lambda x: 'very short' if (x>=0 and x<10) else ( 'short' if(x>=10 and x<50) else ('long' if(x>=50 and x<300) else 'very long')))

Conclusions¶

Research Question 1 (Which genres are most popular from year to year?)¶

>

Different Genres have gained popularity from time to time, I have divided the most popular Genres for '4' time periods -:

  1. Between 1960-1980 - 'Adventure' and 'Family' were the most popular genres

  2. Between 1981-2000 - 'Adventure' was still popular but 'Science Fiction' gained poplarity

  3. Between 2001-2010 - Along with 'Adventure', 'Fantasy' gained popularity

  4. Between 2011-2015 - 'Western' gained popularity along with other genres like 'Action', 'Adventure' being popular

Research Question 2 (What kinds of properties are associated with movies that have high revenues ?)¶

>

In this I have explored effect of '4' different parameters on Adjusted Revenues i.e. 'Release Year', 'Popularity', 'Run Type'(i.e. Run time duration) and 'Genres'.

  1. 'Release Year' - The plots shows the adjusted revenues has decreased over time.

  2. 'Popularity' - Movies with 'Maximum' Popularity does not have the 'Maximum' revenues

  3. 'Run Type' - Movies with Long duration(50-299) bring in the highest revenues, 'Very Short' movies has minimum revenues

  4. 'Genres' - 'Adventure' genre has the highest revenue followed by 'Science Fiction'

Research Question 3 (What is the impact of runtime on popularity?)¶

>

As shown by the bar graph, Movies with long duration(50-299) are the most poular movie type among the fans, followed by short movies type.

Very Short movie type(0-10) are very less popular among the fans.
